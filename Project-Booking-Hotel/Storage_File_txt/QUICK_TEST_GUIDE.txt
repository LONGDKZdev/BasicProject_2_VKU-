================================================================================
                        TROUBLESHOOTING
================================================================================

PROBLEM: "Supabase unavailable, using local seed data"
SOLUTION:
  1. Check internet connection
  2. Check Supabase status: https://status.supabase.com
  3. Check VITE_SUPABASE_URL in .env
  4. Check VITE_SUPABASE_ANON_KEY in .env

PROBLEM: "Image failed to load from Supabase"
SOLUTION:
  1. Check bucket "hotel-rooms" exists
  2. Check bucket is PUBLIC
  3. Check image files are uploaded
  4. Check file names: 1.png, 1-lg.png, etc.
  5. Try URL directly in browser

PROBLEM: "No rooms showing"
SOLUTION:
  1. Check Supabase database has rooms
  2. Check SQL scripts were run
  3. Check console for errors
  4. Check network requests

PROBLEM: "Wrong images showing"
SOLUTION:
  1. Check room type codes (STD, DLX, SUI, PEN, CMB)
  2. Check image mapping in supabaseStorageUrls.js
  3. Verify image file names
